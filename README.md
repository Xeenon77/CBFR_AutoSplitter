# CBFR_AutoSplitter
...is semi-customizable autosplitter for the CBFR of GTAO. By tracking the color change on your screen, it detects when certain segment ends. Thus compared to mosted autosplitters it does not read your memory and solely relies on visual cues.

# Setup

To use the **"death's Autosplitter"** thing on your desktop or wherever to ez open the thing.
